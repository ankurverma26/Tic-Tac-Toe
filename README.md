# Tic-Tac-Toe

A simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

## ✨ Features

- Two-player gameplay
- Winner detection
- Reset game
- Responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 🚀 How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
